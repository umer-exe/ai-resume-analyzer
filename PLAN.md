# Project Plan — AI Powered Resume and Profile Analyzer

## Phase 1, Basic Structure
Goal: Create folders, placeholder files, README, PLAN.md, and .gitignore.

## Phase 2, Flask Backend Starter
Goal: Set up Flask, virtual environment, basic GET / route, and test backend.

## Phase 3, Dummy Analyze API
Goal: Create POST /analyze that accepts profile_text and returns dummy JSON.

## Phase 4, Next.js Frontend Starter
Goal: Create Next.js frontend with one page, textarea, analyze button, and result display.

## Phase 5, Connect Frontend to Backend
Goal: Send profile_text from Next.js to Flask and display dummy response.

## Phase 6, Rule Based Analyzer
Goal: Add basic skill detection, ATS score, strengths, weaknesses, role recommendations, and roadmap.

## Phase 7, Resume Upload
Goal: Add PDF, DOCX, and TXT upload support using PyMuPDF and python-docx.

## Phase 8, UI Polish
Goal: Improve dashboard design, result cards, spacing, and presentation readiness.

## Phase 9, Optional AI Upgrade
Goal: Add Gemini or OpenAI API later for smarter analysis if time allows.
